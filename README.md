# Customer Shopping Behavior Analysis

## Project Overview

This project analyzes customer shopping behavior using Python, SQL, and Power BI. The objective is to identify customer purchasing patterns, uncover business insights, and present findings through an interactive dashboard and business report.

The project follows a complete data analytics workflow, including data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis, dashboard development, and business reporting.

---

## Dataset

**Dataset Name:** Customer Shopping Behavior Dataset

**Description:**
The dataset contains customer transaction and shopping-related information, including demographic details, purchase history, product categories, spending patterns, and other relevant attributes.

**Key Features:**

* Customer ID
* Age
* Gender
* Product Category
* Purchase Amount
* Payment Method
* Shopping Frequency
* Location
* Review Rating

## Project Workflow

### 1. Data Loading

* Imported dataset into Python using Pandas.
* Performed initial inspection of rows, columns, and data types.

### 2. Exploratory Data Analysis (EDA)

* Analyzed dataset structure and distributions.
* Examined customer demographics.
* Identified spending patterns and category preferences.
* Generated visualizations to understand trends and relationships.

### 3. Data Cleaning

* Checked for missing values.
* Removed duplicate records.
* Corrected data inconsistencies.
* Converted columns to appropriate data types.
* Prepared dataset for analysis.

### 4. PostgreSQL & SQL Analysis

* Created a PostgreSQL database.
* Imported cleaned data into PostgreSQL.
* Wrote SQL queries to answer business questions.
* Performed aggregations, filtering, grouping, and trend analysis.

### 5. Dashboard Development

* Connected Power BI to PostgreSQL.
* Created interactive visualizations and KPIs.
* Designed dashboard pages for business stakeholders.
* Enabled filtering and drill-down analysis.

### 6. Reporting

* Documented methodology and findings.
* Summarized key business insights.
* Created a presentation using Gamma AI and PowerPoint.

## How to Run the Project

### Prerequisites

Install the following:

* Python 3.x
* PostgreSQL
* Power BI Desktop
* Jupyter Notebook

### Steps

1. Clone the repository.

```bash
git clone <repository-link>
```

2. Install required Python libraries.

```bash
pip install pandas numpy matplotlib seaborn sqlalchemy psycopg2
```

3. Open the Jupyter Notebook and run the data cleaning and EDA scripts.

4. Create a PostgreSQL database and import the cleaned dataset.

5. Execute the SQL queries provided in the `sql` folder.

6. Open the Power BI dashboard file (`.pbix`) and refresh the data connection.

7. Review the dashboard, report, and presentation for insights.
